# PDA and CFG Simulator

## Features
- Step-by-step PDA simulation
- Stack visualization
- Diagrammatic representation of PDA
- CFG simplification:
  - Remove ε-productions
  - Remove unit productions
  - Remove useless symbols

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
Open index.html in a browser.
